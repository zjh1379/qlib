# 日内执行择时 P1 · 结果 + 判定 (2026-06-04)

**TL;DR:执行择时否决(第 5 连负结果)。** 离线检验"把 factor-2model top-5 的进场从次日开盘价换成日内择时价"。在可信的同口径对比下,**所有可落地的进场规则(vwap / vwap_am / low_band / gap_cond)都跑输次日开盘基线**(净CAGR +22~29% vs 基线 +38%,逐年也几乎全输)。唯一"跑赢"的 `first30_low`(+148%)是**前视偏差**(按当日前30分钟的*已实现*最低价成交,需事后才知道,不可执行),已剔除。结论与本仓统一理论一致:**edge 是日频均值反转,标的从开盘就反弹,开盘价已是接近最优的进场点**;任何"等更便宜"的日内择时都会在最关键的赢家日多付钱。**不进 P2 实时。** 口径 fixed/持5/5天/top_k=5,¥10万隐含,10bp/笔成本,OOF 2020-07→2025-12,免费 baostock 5min。

## 数据可得性 (Task 0 spike + 全量)
- baostock 5min 可用,字段 `time,open,high,low,close,volume,amount`,`adjustflag="3"`(不复权);每日 **48 根**(09:30–11:30 + 13:00–15:00),`time` 取前 14 位解析,首根 09:35 = 开盘段;**历史回到 2020**,覆盖 OOF 全程。
- **数据质量坑(已加固)**:① 偶发**首根/整日全零帧**(停牌/无数据)→ 模拟器判 `has_data`(volume>0 且 open.max>0),无数据 → **回退 open(mult=1.0)计入 `n_fallback`** 而非丢弃;② 偶发**零/坏价 bar**会污染 vwap/最低价 → `entry_rules` 计算时**过滤非正价/零量 bar**;③ 残余坏价 → 模拟器对**进场乘数加合理带 [0.5,1.5]**,越界判 `n_glitch` 回退 open。三层加固后 vwap 的虚高(曾 +118%、2024 +2642%)回落到 +24%(2024 +36.5%)。
- **网络鲁棒**:baostock 长时间顺序查询会断 socket。fetch 层改为**校验 login/query 的 error_code、断连即重登重试、仅在成功时落缓存**(失败不缓存,可断点续跑)。全量 ~1335 个 name-day 拉取 0 报错。

## 口径与方法
- 冠军不变:factor-2model = `_rebuild_2model(oof_lgbmfac, oof_2model)`,选股 fixed/持5/5天/top_k=5(与 +31% 引擎同选股)。
- **进场乘数法(复权无关)**:规则输出"规则价 / 当日开盘"的同日比值(复权不变),模拟器乘到 qlib **复权日线 $open** → 回避复权口径。每笔收益 = `复权open(出场) / (复权open(进场)×乘数) − 1 − 10bp`,5 票等权聚合成周期净收益序列后 cumprod。
- **A股微结构分路**:无数据/停牌 → 回退 open(`n_fallback`);一字封板涨停(全天 low ≥ 涨停价)→ 跳过(`n_unfillable`);`gap_cond` 高开不追 → 跳过(`n_gap_skip`);坏价乘数越界 → 回退 open(`n_glitch`)。
- **对比合法性**:rule 与 open 用**同一聚合、同一组交易/日期**,只差进场价;故"周期 cumprod ≠ 逐日 NAV"造成的绝对量级差(见下锚)在 **rule−open 的 delta 中抵消**——delta 才是研究问题。

## 为什么(与反转理论一致)
模型 edge = **日频超跌反弹**:被选中的票从**开盘就开始反弹**,开盘价≈当日低点。于是:
- **vwap/vwap_am**:全日/上午均价在上涨日**高于开盘**——恰在最赚钱的赢家日多付钱(中位改善仅 +1.4bp 近乎持平,但收益却掉,说明亏在**右尾赢家的非对称**:赢家日 vwap≫open,输家日 vwap<open 省一点,净效应为负)。
- **low_band(限价低于开盘)**:赢家日价格直接走高、限价**不成交**,只能收盘补(更贵);成交的多是继续走弱的输家——典型"接下跌的刀",反 edge。越宽(k=2%)越少触发、越接近基线。
- **gap_cond(高开不追)**:跳过高开反包票=跳过部分强反转赢家 → 跑输。
- **first30_low**:之所以"赢",是因为按*已实现*的 30 分钟最低价成交——**前视**,实盘无法预知该价位下单;其可落地近似就是 low_band(限价已知),而 low_band 跑输。故剔除。

## 不可成交 / 回退占比(结论可信度)
unfillable(涨停)≈ **1.0%**、fallback(无 5min/停牌)≈ **0.7%**、glitch(坏价)≤ **0.1%** —— 三者合计 <2%,远小于规则与基线之间 10~16pp 的差距,**不影响结论**。成交率均 1266~1322 / 1335。

## 诚实判定 / P2 决策门
**P1 = 负结果 → 不进 P2 实时执行。** 没有任何可落地规则在扣费后稳健跑赢"次日开盘"基线;开盘价本身已是反转 edge 的接近最优进场点。与本会话前 4 个杠杆(趋势门 / 分散 / LambdaRank / 个股止损)并列,这是**第 5 个因"对抗/偏离日频均值反转"而被否决的杠杆**——执行择时偏离了"开盘即买反弹"。

**路径不变(再次收窄确认):** 再上台阶只剩**新数据 / 新 alpha**(北向、资金流、行业动量,或日内 5min 用于**重训出新信号**而非择时)——而非在既有日频信号上做执行/风控微操。日内 5min 数据管道(fetch + 缓存 + 鲁棒重试)已就绪,后续若做"日内重训新因子"可直接复用。
