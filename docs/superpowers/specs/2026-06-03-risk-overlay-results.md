# P3 风控暴露层 — 结果（2020-07 → 2025-12, 5.5年）

**Date:** 2026-06-03  **数据:** `oof_2model_2021_2026.pkl`（LGBM+ALSTM），组合 fixed/持5只/5天换，¥10万，最低5元成本。
**方法:** universe 等权尾部收益→合成大盘代理→`compute_exposure(trend_ma)`→引擎滞后暴露（无前视，27 测试全过）。

## 1. 头条
趋势暴露层把**最大回撤腰斩**（−56.7% → −28.2%，达到 ~−25% 目标），代价是净收益 +19%→+11%，**各 regime 净 IR 仍全为正**。

| 配置 | 净CAGR | 最大回撤 | net_ir | Calmar |
|---|---|---|---|---|
| 无 overlay（基线） | +19.0% | −56.7% | 0.673 | 0.34 |
| **选中 ma60 / min0 / 无vt** | **+11.0%** | **−28.2%** | 0.571 | 0.39 |
| 备选 ma200 / min0.2 / 无vt | +16.6% | −37.1% | 0.705 | **0.45** |

选中配置各 regime 净 IR：2020-21 **+1.86** / 2021-22 **+0.55** / 2023-26 **+0.11**（全正）。

## 2. 全扫描（ma ∈ {60,120,200} × min_exposure ∈ {0,0.2} × vol_target ∈ {无,0.15}）
- **ma 越短（60）→ 回撤越低**（−28%），但收益也越低（+11%）；**ma 越长（200）→ 收益更高（+15~17%）但回撤更大（−37~40%）**。趋势窗口就是回撤/收益的旋钮。
- **vol_target(0.15) 一律有害**：降收益、几乎不降回撤（A股阴跌型熊市波动不一定先飙）→ 弃用。
- **min_exposure**：0（可空仓）回撤最低；0.2 floor 略升回撤、有时升收益。
- 全部配置 Calmar 在 0.18–0.45，**选中 0.39 / 最佳 0.45**，均优于无 overlay 的 0.34。

## 3. 验收（spec §9）
1. 最大回撤 ≈ −25%（−20~−30%）：✅ 选中 −28.2%（在带内）。
2. 净 CAGR > +10% 目标：✅ +11.0%。
3. 各 regime 净 IR 为正：✅。
4. Calmar 明显改善：✅ 0.34 → 0.39（备选 0.45）。
5. 无前视 + 成本测试全过：✅（27 测试）。

## 4. 诚实评价
- ✅ 风控目标达成：回撤腰斩到 ~目标，各 regime 仍正。
- ⚠️ **权衡很陡**：回撤减半的同时收益也几乎减半（19%→11%），Calmar 仅 0.34→0.39（中等）。这说明暴露择时主要是"削峰填谷"，没有创造新 alpha。
- 想要"既低回撤又高收益"（Calmar 上 0.5+），靠的是**更厚的 alpha**（P2b 短线因子）或**更聪明的风控**（个股止损/行业中性/动态仓位），而非单纯的大盘择时。
- **可落地推荐**：目标 −25% 用 ma60/min0/无vt（+11%/−28%）；若接受 −37% 换更高收益+更优 Calmar，用 ma200/min0.2（+16.6%/−37%/0.45）。

## 5. 后续
- P2b 短线因子（加厚 alpha，改善整条前沿）。
- 个股止损 / 行业中性（进一步压回撤而少损收益）。
- 修 TRA 补第三模型。
- 把"持5只/5天换 + 暴露层"落进产品。
